Webmapping

Les 3: Web Maps

In de derde les zijn we bezig geweest met webmapping. We hebben webkaarten van MapLibre en Leaflet uitgeprobeerd. Het meest waardevolle wat ik hier mocht leren van Niene is hoe API's werken.

Blijkbaar is de API een manier om met een server te praten en als hij terugpraat, kan je dat gebruiken om informatie op te halen van de server en te tonen op je website. Ook belangrijk is, dat je dus openbare en gesloten API's hebt. Waarbij gesloten API's ook via een creditssysteem kunnen werken. Hierbij betaal 1 credit per actie van de API.

In deze les hebben we geëxperimenteerd met verschillende kaartbibliotheken en leerden we hoe we interactieve kaarten kunnen inbedden in webpagina's. Dit opende een hele nieuwe wereld van mogelijkheden voor het visualiseren van geografische data op het web.
