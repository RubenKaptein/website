Webmapping



Html, css en javascript

Webmapping is een engelse term voor het proces van maken, designen en opleveren van een kaart op het internet. Denk bijvoorbeeld aan Google Maps. Om een webmap te kunnen bouwen, heb je drie onderdelen nodig.
1. GIS: de kaart
2. Het web: HTML, CSS, Javascript and API's om de kaart te renderen.
3. Data bronnen: de informatie die je wil laten zien op de kaart.

Als voorbeeld, zie je hieronder twee kaarten in deze website als demonstratie.
